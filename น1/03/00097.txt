<span class="LineNumber"> 002 </span>                             แห่งกาสี                ชื่อพาราณสี<SUP>๒</SUP>
<span class="LineNumber"> 003 </span>                             แห่งโกสละ             ชื่อสาวัตถี<SUP>๓</SUP>
<span class="LineNumber"> 004 </span>                             แห่งวัชชี                 ชื่อเวสาลี<SUP>๔</SUP>
<span class="LineNumber"> 005 </span>                             แห่งมัลละ               ชื่อเดิมกุสาวดี<SUP>๕</SUP> 
<span class="LineNumber"> 006 </span>แต่ภายหลังแยกเป็น  กุสินารา  กับ  ปาวา.

<span class='footnote'>๑.  มคธะ ตั้งอยู่ใต้แม่น้ำคงคาตองกลาง เทียบราวมณฑลพิหาร ( Bihar ) มีแม่น้ำ</span>
<span class='footnote'>คงคาอยู่ทิศเหนือและมีภูเขาวินธยะ ( Vindhya ) อยู่ทิศใต้และทิศตะวันตก นครเก่า</span>
<span class='footnote'>ชื่อ คิริพพชะ ตั้งอยู่บนเนินเขา ห่างจากปัตนะทางทิศใต้เฉียงอาคเนย์ ๖๕ กิโลเมตร</span>
<span class='footnote'>ภายหลังพระเจ้าพิมพิสารทรงสร้างนครใหม่ชื่อราชคฤห์ ตั้งอยู่ที่เชิงเขา ต่อมาพระเจ้า</span>
<span class='footnote'>อุทายีราชนัดดาของพระเจ้าอชาตศัตรูได้ย้ายไปตั้งที่นครปาฏลิบุตร บัดนี้เรียกว่า ปัตนะ</span>
<span class='footnote'>( Patna ) เหนือแม่น้ำโสน ระหว่างแม่น้ำนี้กับอังคะ.   ๒.  กาสี ตั้งอยู่ตรงที่บรรจบ</span>
<span class='footnote'>แม่น้ำคงคาและยมุนา นครหลวงชอ พาราณสี บัดนี้เรียกว่า เพนาเรส ( Benares ).</span>
<span class='footnote'>๓.  โกสละ  ตั้งอยู่ระหว่างภูเขาหิมาลัยกับแม่น้ำคงคาตอนกลาง เทียบราวมณฑลอยุธยา</span>
<span class='footnote'>คือโอธ ( Oudh ) บัดนี้ทิศเหนือจดภูเขาเนปาล ( Nepal ) ทิศตะวันตกและทิศใต้จด</span>
<span class='footnote'>แม่น้ำคงคา ทิศตะวันออกจดแคว้นกาสีต่อกับมคธะ นครหลวงชื่อสาวัตถี บัดนี้เรียกว่า</span>
<span class='footnote'>สะเหต-มะเหต ( Sahet-mahet ) ตั้งอยู่ ณ ฝั่งลำนำน้ำรับติ ( Rubdi ) ริมเชิงเขาห่าง</span>
<span class='footnote'>จากเขตโอธราว ๘๐ กิโลเมตร. และทางทิศใต้ห่างจากแดนเนปาลราว ๕๐ กิโลเมตร.</span>
<span class='footnote'>๔.  วัชชี  ตั้งอยู่บนฝั่งทิศตะวันออกแห่งลำน้ำคันธกะติดต่อกับมัลละ นครหลวงชื่อ</span>
<span class='footnote'>เวสาลี บัดนี้เรียกว่า แขวงเพสารห์ ( Besarh ) ห่างจากปัตนะราว ๔๔ กิโลเมตร.</span>
<span class='footnote'>๕.  มัลละ  ตั้งอยู่ถัดโกสละมาทางทิศตะวันออก อยู่ทิศเหนือแคว้นวัชชี และทิศตะวัน</span>
<span class='footnote'>ออกแห่งสักกะ นครหลวงชื่อ กุสินคร หรือกุสินารา ตั้งอยู่ที่ประชุมลำน้ำรับดิกับลำน้ำ</span>
<span class='footnote'>คันธกะ อยู่เหนือเนปาล บัดนี้เรียกแขวงกาเซีย ( Kasia ) ส่วนนครปาวาอยู่ระหว่าง</span>
<span class='footnote'>นครกุสินารากับนครเวสาลี บัดนี้เรียกแขวงปัทระโอนะ ( Padraona ).</span>
