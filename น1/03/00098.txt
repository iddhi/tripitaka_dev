<span class='footnote'>๑.  เจตี  ตั้งอยู่ต่อแคว้นอวันตี ( มาลวะ ) ออกมาทางทิศตะวันออกเฉียงใต้ นครหลวง</span>
<span class='footnote'>ชื่อโสตถิวดี.   ๒.  วังสะ  ตั้งอยู่ใต้ลำน้ำยมุนา ทางทิศใต้แคว้นโกสละและทางทิศ</span>
<span class='footnote'>ตะวันตกแคว้นกาสี นครหลวงชื่อ โกสัมพี ตั้งอยู่เหนือฝั่งใต้ลำน้ำยมุนา ที่เรียก</span>
<span class='footnote'>ว่าลำน้ำโกสัม อยู่ทางทิศหรดีในตำบลอัลลาฮาบัด ( Allahabad ).   ๓.  ในรัฏฐ-</span>
<span class='footnote'>ปาลสูตร มัชฌิมนิกาย มัชฌิมปัณณาสก์ กล่าวว่า ถุลลโกฏฐิตนิคมเป็นราชสำนัก</span>
<span class='footnote'>ของพระเจ้ากุรุ. ว. ว.  กุรุ  ตั้งอยู่ตรงลุ่มน้ำยมุนาตอนบน เทียบราวมณฑลปัญจาป</span>
<span class='footnote'>( Panjap ) และเมืองเดลฮี ( Delhi ) นครหลวงอินเดียบัดนี้.   ๔.  ในอรรถกถาชาดก</span>
<span class='footnote'>กัมปิลละ เป็นชื่อรัฏฐะ ปัญจาละ เป็นชื่อนคร กลับกันไป. ว. ว.  ปัญจาละ ตั้งอยู่</span>
<span class='footnote'>ตรงลุ่มน้ำคงคาตอนบน มีแคว้นโกศลอยู่ทิศตะวันออก แคว้นกุรุอยู่ทิศตะวันตก ภูเขา</span>
<span class='footnote'>หิมาลัยอยู่ทิศเหนือและแม่น้ำคงคาอยู่ทิศใต้ เทียบราวมณฑลอกราบัดนี้ นครหลวงเดิม</span>
<span class='footnote'>ชื่อหัสตินาปุระหรือหัสดิน ต่อมาจึงแยกไปตั้งนครหลวงใหม่ ๒ แห่ง คือกัมปิลละ ตั้ง</span>
<span class='footnote'>อยู่ฝั่งเหนือแม่น้ำคงคา ถัดลงมาถึงสังกัสสนครแล้วจึงถึงกันยากุพช์ ซึ่งบัดนี้เรียกว่า</span>
<span class='footnote'>กระเนาช์ ( Kanounge )   ๕.  สาคละ เป็นนครหลวงของมัททชนบท ตามอรรถกถา</span>
<span class='footnote'>ชาดก มัจฉะกับมัททชนบท รวมเป็นอาณาจักรเดียวกัน น่าจะเห็นว่าชนบทเดียวกัน</span>
<span class='footnote'>ด้วย ว. ว.  ๕. มัจฉะ ตั้งอยู่ระหว่างลำน้ำสินธูกับยมุนาตอนบน มีแคว้นโกศลอยู่ทิศ</span>
<span class='footnote'>ตะวันออก มีแคว้นสุรเสนะอยู่ทิศเหนือ มีแคว้นกุรุอยู่ทิศใต้ มีมณฑลปัญจาปอยู่ทิศ</span>
<span class='footnote'>พายัพ นครหลวงชื่อสาลคะ ตั้งอยู่เหนือแม่น้ำอสิคนีหรือจันทรภาค.   ๖. สุรเสนะ ตั้ง</span>
<span class='footnote'>อยู่ระหว่างลำน้ำสินธูกบยมุนาตอนล่าง ราวมณฑลราชปุตต์ฐาน ( Rajputthan ) บัดนี้</span>
<span class='footnote'>นครหลวงเก่าซึ่งเป็นราชธานีของพระกฤษณะ ชื่อมถุรา ราวเมืองมัตตรา ( Mattra )</span>
