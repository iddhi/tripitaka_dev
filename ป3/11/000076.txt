<span class="LineNumber"> 001 </span>                                        เรื่องสัมพหุลภิกษุ
<span class="LineNumber"> 002 </span>           ๑๖.  ๕๗/๑๕  ตั้งแต่  เตสุ  กิร  เอกจฺจานํ  เอวํ  อโหสิ
<span class="LineNumber"> 003 </span>เป็นต้นไป.  
<span class="LineNumber"> 004 </span>          กิร  ได้ยินว่า  เอวํ  จินฺตนํ  อ. ความคิดอย่างนี้ว่า  มยํ  อ.  เรา  ท.
<span class="LineNumber"> 005 </span>สมฺปนฺนสีลา  เป็นผู้มีศีลอันถึงพร้อมแล้ว  ( อมฺห )  ย่อมเป็น  มยํ
<span class="LineNumber"> 006 </span>อ.  เรา  ท.  ธูตงฺคธรา  เป็นผู้ทรงไว้ซึ่งองค์เป็นเครื่องขจัด  ( อมฺห )
<span class="LineNumber"> 007 </span>ย่อมเป็น  มยํ  อ. เรา  ท.  พหุสฺสุตา  เป็นผู้มีพระพุทธพจน์อันฟังแล้วมาก
<span class="LineNumber"> 008 </span>( อมฺห )  ย่อมเป็น  มยํ  อ.  เรา ท.  ปนฺตเสนาสนวาสิโน  เป็นผู้มีปกติ
<span class="LineNumber"> 009 </span>อยู่ในเสนาสนะอันสงัดแล้ว  ( อมฺห )  ย่อมเป็น มยํ  อ.  เรา ท.
<span class="LineNumber"> 010 </span>ฌานลาภิโน  เป็นผู้มีปกติได้ซึ่งฌาน  ( อมฺห ) ย่อมเป็น  อรหตฺตํ
<span class="LineNumber"> 011 </span>อ.  ความเป็นแห่งพระอรหันต์  อมฺหากํ  ทุลฺลภํ เป็นคุณชาติอันเรา  ท.
<span class="LineNumber"> 012 </span>ได้โดยยาก  ( โหติ )  ย่อมเป็น น  หามิได้  มยํ  อ.  เรา  ท.  ปาปุณิสฺสาม
<span class="LineNumber"> 013 </span>จักบรรลุ  อรหตฺตํ  ซึ่งความเป็นแห่งพระอรหันต์ ( อมฺหากํ )  อิจฺฉิต-
<span class="LineNumber"> 014 </span>ทิวเสเอว  ในวันแห่งเรา  ท.  ปรารถนาแล้วนั่นเทียว  อิติ  ดังนี้  อโหสิ
<span class="LineNumber"> 015 </span>ได้มีแล้ว เตสุ  ภิกฺขูสุ ในภิกษุ  ท.  เหล่านั้นหนา  เอกจฺจานํ  ภิกฺขูนํ
<span class="LineNumber"> 016 </span>แก่ภิกษุ  ท.  บางพวก  ฯ  ตตฺถ  ภิกฺขูสุ  ในภิกษุ  ท.  เหล่านั้นหนา
<span class="LineNumber"> 017 </span>เยปิ  ภิกฺขู  อ.  ภิกษุ  ท.  แม้เหล่าใด  อนาคามิโน  เป็นพระอนาคามี
<span class="LineNumber"> 018 </span>( โหนฺติ )  ย่อมเป็นเอตํ  จินฺตนํ  อ. ความคิดนี้ว่า  อิทานิ  ในกาลนี้
<span class="LineNumber"> 019 </span>อรหตฺตํ  อ. ความเป็นแห่งพระอรหันต์  อมฺหากํ ทุลฺลภํ  เป็นคุณชาติ
