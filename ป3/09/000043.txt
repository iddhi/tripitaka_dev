<span class="LineNumber"> 002 </span>           ๑๐.  ๕๓/๑๕  ตั้งแต่  สาวตฺถิยํ  กิเรกสฺมึ  อุโปสถทิวเส
<span class="LineNumber"> 003 </span>เป็นต้นไป.  
<span class="LineNumber"> 004 </span>           กิร  ได้ยินว่า  เอกสฺมึ  อุโปสถทิวเส  ในวันแห่งอุโบสถวันหนึ่ง
<span class="LineNumber"> 005 </span>อิตฺถิโย  อ.  หญิง  ท.  ปฺจสตมตฺตา  ผู้มีร้อยห้าเป็นประมาณ  สาวตฺถิยํ
<span class="LineNumber"> 006 </span>ในเมืองสาวัตถี  อุโปสถิกา  เป็นผู้รักษาซึ่งอุโบสถ  หุตฺวา  เป็น
<span class="LineNumber"> 007 </span>อคมํสุ  ได้ไปแล้ว  วิหารํ  สู่วิหาร ฯ  วิสาขา  อ.  นางวิสาขา
<span class="LineNumber"> 008 </span>อุปสงฺกมิตฺวา  เข้าไปหาแล้ว  ตาสุ  อิตฺถีสุ  ในหญิง  ท.  เหล่านั้นหนา
<span class="LineNumber"> 009 </span>มหลฺลกิตฺถิโย  ซึ่งหญิงผู้แก่  ท.  ปุจฺฉิ  ถามแล้วว่า  อมฺมา  แน่ะ
<span class="LineNumber"> 010 </span>แม่ ท.  ตุมฺเห  อ. ท่าน  ท.  อุโปสถิกา  เป็นผู้รักษาซึ่งอุโบสถ  ชาตา
<span class="LineNumber"> 011 </span>เกิดแล้ว  อตฺถ  ย่อมเป็น  กิมตฺถํ  เพื่ออะไร  อิติ  ดังนี้  ( วจเน )
<span class="LineNumber"> 012 </span>ครั้นเมื่อคำว่า  ( มยํ )  อ. ดิฉัน  ท.  ปตฺเถตฺวา  ปรารถนาแล้ว
<span class="LineNumber"> 013 </span>ทิพฺพสมฺปตฺตึ  ซึ่งสมบัติอันเป็นทิพย์  ( อุโปสถิกา )  เป็นผู้รักษาซึ่ง
<span class="LineNumber"> 014 </span>อุโบสถ  ( ชาตา )  เกิดแล้ว  ( อมฺห )  ย่อมเป็น  อิติ  ดังนี้  ตาหิ
<span class="LineNumber"> 015 </span>มหลฺลกิตฺถีหิ  อันหญิงผู้แก่  ท.  เหล่านั้น  วุตฺเต  กล่าวแล้ว  ปุจฺฉิ
<span class="LineNumber"> 016 </span>ถามแล้ว  มชฺฌิมิตฺถิโย  ซึ่งหญิงผู้มีในท่ามกลาง  ท.  ( วจเน )  ครั้น
<span class="LineNumber"> 017 </span>เมื่อคำว่า  ( มยํ )  อ. ดิฉัน ท.  ( อุโปสถิกา )  เป็นผู้รักษาซึ่งอุโบสถ
<span class="LineNumber"> 018 </span>( ชาตา )  เกิดแล้ว  มุจฺนตฺถาย  เพื่อต้องการแก่อันพ้น  สปตฺติวาสา
<span class="LineNumber"> 019 </span>จากการอยู่ร่วมด้วยหญิงผู้เป็นไปกับด้วยผัว  ( อมฺห )  ย่อมเป็น  อิติ
