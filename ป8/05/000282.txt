<span class="LineNumber"> 001 </span>คือ ความเป็นใหญ่ ( อิสสริยะ ) ธรรม ยศ สิริ ความได้อย่างปรารถนา  
<span class="LineNumber"> 002 </span>( กามะ ) และความพยายาม ( ปยัตตะ ) ก็ความเป็นใหญ่ในพระจิตของ
<span class="LineNumber"> 003 </span>พระองค์เองอย่างยอดเยี่ยมมีอยู่ หรือความเป็นใหญ่ที่โลกสมมติกัน 
<span class="LineNumber"> 004 </span>( ๘ อย่าง ) มีทำร่างกายให้เล็ก ( อณิมา )และทำร่างกายให้เบา 
<span class="LineNumber"> 005 </span>( เดินอากาศได้ ลังฆิมา ) เป็นต้น อันบริบูรณ์ด้วยอาการทั้งปวงของ
<span class="LineNumber"> 006 </span>พระองค์ก็มีอยู่โดยนัยเดียวกันนั้น พระธรรมอันเป็นโลกุตตระของ
<span class="LineNumber"> 007 </span>พระองค์ก็มีอยู่ พระยศของพระองค์ปรากฏทั่วโลก ๓ เป็น
<span class="LineNumber"> 008 </span>พระยศที่ทรงได้โดยพระคุณตามที่เป็นจริง เป็นพระยศที่บริสุทธิ์ยิ่งนัก
<span class="LineNumber"> 009 </span>ก็มีอยู่ พระสิริ ( ความงามสง่า ) แห่งพระอังคาพยพใหญ่น้อยทั้งปวง
<span class="LineNumber"> 010 </span>บริบูรณ์ด้วยอาการทุกสิ่งทุกอย่าง สามารถให้เกิดขวัญตาและขวัญใจ
<span class="LineNumber"> 011 </span>แก่ชนผู้ขวนขวายในการเฝ้าชมพระรูปกาย ก็มีอยู่ พระกามะ อันหมาย
<span class="LineNumber"> 012 </span>ถึงความสำเร็จแห่งประโยชน์ที่ทรงพระประสงค์ ก็มีอยู่ เพราะประโยชน์
<span class="LineNumber"> 013 </span>ใดๆที่พระองค์มีพระประสงค์แล้ว ทรงปรารถนาแล้ว จะเป็น
<span class="LineNumber"> 014 </span>ประโยชน์ตนก็ตาม ประโยชน์ผู้อื่นก็ตาม ประโยชน์นั้นๆก็สำเร็จ
<span class="LineNumber"> 015 </span>สมพระประสงค์ทั้งนั้น และพระปยัตตะ กล่าวคือสัมมาวายามะ อัน
<span class="LineNumber"> 016 </span>เป็นเหตุเกิดขึ้นแห่งความเป็นครูของโลกทั้งปวง ก็มีอยู่ เพราะเหตุนั้น 
<span class="LineNumber"> 017 </span>บัณฑิตจึงถวายพระนามว่า ภควา โดยอรรถ ( วิเคราะห์ ) นี้ว่า ภคา
<span class="LineNumber"> 018 </span>อสฺส สนฺติ - ทรงมีภคธรรม เพราะความที่ทรงประกอบด้วยภคธรรม
<span class="LineNumber"> 019 </span>ทั้งหลาย ( ที่กล่าวมา ) นี้ ประการ ๑ 
<span class="LineNumber"> 020 </span>                             [แก้บท วิภตฺตวา] 
<span class="LineNumber"> 021 </span>          อนึ่ง เพราะเหตุที่พระผู้มีพระภาคเจ้าทรงเป็น <B>วิภตฺตวา</B> มีคำ
