<span class="LineNumber"> 002 </span>          กรรมฐานมี ๒ อย่าง คือ สัพพัตถกกัมมฐาน ๑ ปาริหาริย- 
<span class="LineNumber"> 003 </span>กัมมฐาน ๑ ในกรรมฐาน ๒ อย่างนั้น ( กรรมฐานคือ ) เมตตา ( อัน
<span class="LineNumber"> 004 </span>แผ่ไป ) ในประชาชนต่างๆ มีหมู่ภิกษุเป็นต้น ก็ดี มรณสติก็ดี ชื่อว่า 
<span class="LineNumber"> 005 </span>สัพพัตถกกัมมฐานอาจารย์พวกหนึ่งกล่าวว่า อสุภสัญญา ( เป็น 
<span class="LineNumber"> 006 </span>สัพพัตถกกัมมฐาน ) ก็มี จริงอยู่ เมตตา ( นั้น ) ขั้นแรก ภิกษุผู้
<span class="LineNumber"> 007 </span>บำเพ็ญกรรมฐาน พึงกำหนดเอาหมู่ภิกษุนั้นแล้ว เจริญไปในหมู่ภิกษุ
<span class="LineNumber"> 008 </span>ผู้อยู่ในสีมา ( เดียวกันกับตน )โดยนัยว่า ขอภิกษุผู้อยู่ในสีมาทั้งหลาย 
<span class="LineNumber"> 009 </span>จงเป็นผู้มีความสุข    ไม่เบียดเบียนกันเถิด ต่อนั้นจึงเจริญไปในหมู่
<span class="LineNumber"> 010 </span>เทวดาผู้อยู่ในสีมา ( นั้น ) แต่นั้นจึงเจริญไปในหมู่อิสรชนในโคจรคาม 
<span class="LineNumber"> 011 </span>แล้วจึงเจริญไปในคน ( ชาวบ้าน )<SUP>๑</SUP> ในโคจรคามนั้น ตลอดไปถึงสัตว์ 
<span class="LineNumber"> 012 </span>( ดิรัจฉาน ) ทั้งปวง ก็แล ด้วยเมตตา ( ที่เธอเจริญไป ) ในหมู่ภิกษุ
<span class="LineNumber"> 013 </span>เธอย่อมทำให้ภิกษุทั้งหลายผู้อยู่ร่วมกันเกิดมีจิตอ่อนโยน ( ในตัวเธอ
<span class="LineNumber"> 014 </span>ได้ )<SUP>๒</SUP>    เมื่อเป็นเช่นนั้น   ภิกษุเหล่านั้นก็เป็นสุขสังวาส   ( มีการอยู่
<span class="LineNumber"> 015 </span>ร่วมกันเป็นสุข)สำหรับเธอ ด้วยเมตตา ( ที่เจริญไป ) ในหมู่เทวดา
<span class="LineNumber"> 016 </span>ผู้อยู่ในสีมานั้น เธอจะเป็นผู้ที่เทวดาทั้งหลายผู้มีจิตอันเมตตานั้น
<span class="LineNumber"> 017 </span>ทำให้อ่อนโยนแล้ว จัดการรักษาอย่างดี โดยการรักษาที่เป็นธรรม
<span class="LineNumber"> 018 </span>ด้วยเมตตา ( ที่เจริญไป ) ในอิสรชนในโคจรคาม เธอจะเป็นผู้ที่
