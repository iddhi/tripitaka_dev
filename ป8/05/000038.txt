<span class="LineNumber"> 008 </span>ก็ไม่ควรทาเท้าเข้าก่อนทีเดียว เพราะถ้าน้ำมันนั้น จะพึงเป็นน้ำมัน
<span class="LineNumber"> 009 </span>สำหรับชะโลมตัวอาจารย์ไซร้ ก็จะเป็นการไม่สมควรไป เพราะฉะนั้น
<span class="LineNumber"> 010 </span>พึงทาศีรษะก่อน แล้วจึงทาอวัยวะอื่นมีคอเป็นต้น ( ต่อไป ) ต่อท่าน
<span class="LineNumber"> 011 </span>บอกว่า " อาวุโส น้ำมันนี่เป็นน้ำมันสำหรับใช้ได้ทุกอย่าง ทาเท้าก็ได้ " 
<span class="LineNumber"> 012 </span>ดังนี้ จึงแตะที่ศีรษะหน่อยหนึ่ง ต่อนั้นจึงทาเท้า ( เสร็จ ) แล้ว พึงเรียน 
<span class="LineNumber"> 013 </span>( ท่าน ) ว่า จะเก็บกะโหลกน้ำมัน ( ที่ไหน เพื่อท่านจะได้ชี้ที่เก็บให้ 
<span class="LineNumber"> 014 </span>แต่ ) เมื่ออาจารย์รับเอาไป ( จะเก็บเอง ) ก็ให้ ( ท่าน ) ไปเถิด
