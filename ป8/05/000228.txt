<span class="LineNumber"> 001 </span>เสีย  นิมิตอันอ่อนนั้นก็จักเสื่อมไป เมื่อเธอคิดว่าจักไปถือเอานิมิตอีก   
<span class="LineNumber"> 002 </span>แม้เป็นผู้ใคร่จะไป   ก็ไม่อาจแม้แต่จะเข้าไปสู่ป่าช้าได้ เพราะถูก
<span class="LineNumber"> 003 </span>อมนุษย์หรือสัตว์ร้ายขวางเสียบ้าง   ( หากเข้าป่าช้าได้ ) นิมิตก็กลายไป
<span class="LineNumber"> 004 </span>เสียบ้าง   จริงอยู่   อุทธุมาตกอสุภ  คง ( สภาพ ) อยู่วัน ๑ หรือ ๒ วัน 
<span class="LineNumber"> 005 </span>เท่านั้น ก็ถึงความเป็นอสุภอย่างอื่นมีวินีลกอสุภเป็นต้นไปเสีย ใน
<span class="LineNumber"> 006 </span>กรรมฐานทั้งหมด ชื่อว่ากรรมฐานที่หาได้ยากเสมอด้วยอุทธุมาตก-
<span class="LineNumber"> 007 </span>อสุภนั้นไม่มี เพราะเหตุนั้น เมื่อนิมิตเสื่อมไปเสียแล้วอย่างนี้ 
<span class="LineNumber"> 008 </span>ภิกษุนั้นพึงนั่งในที่พักกลางคืนหรือในที่พักกลางวันก็ตาม แล้ว ( ย้อน ) 
<span class="LineNumber"> 009 </span>พิจารณาถึงทางไปทางมา ( ตั้งแต่ต้น ) จนถึงที่ๆ ตนนั่งคู้บัลลังก์ ( อยู่
<span class="LineNumber"> 010 </span>เวลานี้ ) อย่างนี้ว่า " เราออกจากวิหารโดยประตูชื่อนี้ เดินไปทางที่
<span class="LineNumber"> 011 </span>ตรงไปทิศโน้น แล้วถือเอา ( เลี้ยว ) ซ้ายในที่ชื่อโน้น ถือเอาขวา
<span class="LineNumber"> 012 </span>ที่โน่น ในที่แห่งอสุภนั้น ก้อนหินอยู่ตรงโน้น หรือจอมปลวก ไม้ต้น 
<span class="LineNumber"> 013 </span>ไม้กอ ไม้เถา อย่างใดอย่างหนึ่ง อยู่ตรงโน้นๆ เรานั้น ( เข้า ) ไป
<span class="LineNumber"> 014 </span>โดยทางนั้น จึงได้เห็นอสุภในที่ชื่อโน้น ในที่นั้นเรายืนหันหน้าไปทาง
<span class="LineNumber"> 015 </span>ทิศโน้น สังเกตจำนิมิตทั้งหลายโดยรอบอย่างนั้นๆ ถือเอาอสุภนิมิต 
<span class="LineNumber"> 016 </span>( เป็นอุคคหนิมิต ) ขึ้นได้อย่างนั้น แล้ว ( กลับ ) ออกจากป่าช้าทางทิศ
<span class="LineNumber"> 017 </span>โน้น ทำกิจนี้ๆ ตามทางเช่นนั้นๆ พลางมาถึง ( ที่อยู่ )แล้วนั่งอยู่นี่ " 
<span class="LineNumber"> 018 </span>เมื่อเธอ ( ย้อน ) พิจารณาไปอย่างนี้ นิมิตนั้นจะปรากฏขึ้น คือเข้ามา
<span class="LineNumber"> 019 </span>ตั้งอยู่ ( ในคลอง ) ราวกะวางไว้ข้างหน้า ( ต่อนั้น ) กรรมฐานจะดำเนิน
<span class="LineNumber"> 020 </span>ไปสู่วิถีด้วยอาการดุจอาการครั้งแรก เพราะเหตุนั้นท่านจึงกล่าวว่า 
<span class="LineNumber"> 021 </span>การพิจารณาทางไปมา มีประโยชน์คือยังกรรมฐาน ( ที่เสื่อม ) ให้
