<span class="LineNumber"> 001 </span>อัปปนาแล้ว ก็จำต้องขยายในฐานะ ( จุด ) หนึ่งเป็นแน่แท้ เพราะ 
<span class="LineNumber"> 002 </span>เหตุนั้น ข้าพเจ้าจึงกล่าวว่า " พึงขยาย ( ส่วน ) ปฏิภาคนิมิตตามที่ได้ 
<span class="LineNumber"> 003 </span>ไว้แล้ว "
<span class="LineNumber"> 004 </span>          ( ต่อไป ) นี้เป็นนัยในการขยายปฏิภาคนิมิตนั้น พระโยคีนั้น
<span class="LineNumber"> 005 </span>ไม่พึงขยายนิมิตนั้นโดยทำนองขยายบาตร ( ดิน ) ทำนองขยายขนม 
<span class="LineNumber"> 006 </span>ทำนองขยายข้าว ทำนองขยายตัวแห่งเถาวัลย์และทำนองขยายผ้า พึง
<span class="LineNumber"> 007 </span>นึกกำหนด ( ขยาย )ขนาดแห่งนิมิตที่ได้ไว้อย่างไรนั้นว่า ( จงขยาย ) 
<span class="LineNumber"> 008 </span>ประมาณ ๑ องคุลี ๒ องคุลี ๓ องคุลี ๔ องคุลี ... โดยลำดับ แล้วจึงขยาย
<span class="LineNumber"> 009 </span>เท่าที่กำหนดไว้ หากมิได้กำหนด ( ดังกล่าว ) ไม่ควรขยาย เปรียบ
<span class="LineNumber"> 010 </span>เหมือนชาวนา กำหนดที่ๆ จะพึงไถด้วย ( รอย ) ไถแล้วจึงไถไปภายใน
<span class="LineNumber"> 011 </span>ที่กำหนดไว้ ฉะนั้น อนึ่งเหมือนภิกษุทั้งหลายจะผูกสีมา  กำหนด
<span class="LineNumber"> 012 </span>นิมิตทั้งหลายก่อนแล้วจึงผูกภายหลัง ฉะนั้น แต่นั้นเมื่อจะขยาย 
<span class="LineNumber"> 013 </span>( ต่อไป ) ก็พึงกำหนดโดยขนาดคืบ ๑ ศอก ๑ เท่าบริเวณด้านหน้า 
<span class="LineNumber"> 014 </span>( แห่งวิหาร ) เท่าแดนวิหาร และแล้ว เท่าแดนหมู่บ้าน แดนนิคม 
<span class="LineNumber"> 015 </span>แดนชนบท แดนราชอาณาเขต และแดนสมุทร หรือโดยขนาดเท่า
<span class="LineNumber"> 016 </span>จักรวาล หรือแม้ยิ่งกว่านั้น ก็ขยายไปได้ เปรียบเหมือนลูกหงส์<SUP>๑</SUP> 
<span class="LineNumber"> 017 </span>ตั้งแต่กาลที่ ( ขน ) ปีกงอกไป ( มันหัด ) โผไปทีละน้อยๆ กล้าแข็ง
<span class="LineNumber"> 018 </span>เข้า ก็ ( บิน ) ไปสู่ที่ใกล้พระจันทร์พระอาทิตย์ได้โดยลำดับ ฉันใด
<span class="LineNumber"> 019 </span>ภิกษุก็ฉันนั้นเหมือนกัน เมื่อกำหนด ( ขนาด ) แล้วขยายนิมิตไปโดย
<span class="LineNumber"> 020 </span>นัยที่กล่าวแล้ว ก็ย่อมขยายไปจนเท่าขนาดจักรวาล หรือยิ่งกว่านั้นได้
