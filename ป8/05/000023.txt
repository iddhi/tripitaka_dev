<span class="LineNumber"> 001 </span>                                [คณปลิโพธ] 
<span class="LineNumber"> 002 </span>	คณะภิกษุผู้เรียนพระสูตรก็ดี คณะภิกษุผู้เรียนพระอภิธรรมก็ดี 
<span class="LineNumber"> 003 </span>ชื่อว่า คณะ ภิกษุใดมัวให้อุเทศบ้าง ปริปุจฉาบ้างแก่คณะนั้น ย่อม
<span class="LineNumber"> 004 </span>ไม่ได้โอกาสสำหรับสมณธรรม คณะจึงเป็นปลิโพธต่อภิกษุนั้น คณ-
<span class="LineNumber"> 005 </span>ปลิโพธนั้น เธอพึงตัดเสียด้วยอุบายดังต่อไปนี้ ถ้างาน ( เรียน ) ของ
<span class="LineNumber"> 006 </span>ภิกษุเหล่านั้นเสร็จไปแล้วมาก งานที่เหลือน้อยไซร้ เธอพึงทำงานนั้น
<span class="LineNumber"> 007 </span>เสียให้เสร็จแล้วเข้าป่าไปเสียเถิด ถ้างานที่เสร็จแล้วน้อย ที่เหลือมากไซร้ 
<span class="LineNumber"> 008 </span>เธอ ( พึงไปหาคณวาจก ( ผู้สอนหมู่ ) อื่นให้สอนแทน ) อย่างพึงไป 
<span class="LineNumber"> 009 </span>( ไกล ) เลย ๑ โยชน์ เข้าไปหาคณวาจกอื่น ( ซึ่งตั้งสำนักอยู่ ) ภายใน
<span class="LineNumber"> 010 </span>ที่กำหนด ๑ โยชน์ขอร้องให้ท่านช่วยสงเคราะห์ภิกษุในคณะของตน 
<span class="LineNumber"> 011 </span>ด้วยสังคหวิธีมีให้อุเทศเป็นต้น ( แทนตน ) เถิด แม้นไม่ได้อย่างว่า
<span class="LineNumber"> 012 </span>มานี้ ก็พึงบอก ( ภิกษุเหล่านั้น ) ว่า "อาวุโสทั้งหลาย กิจอย่างหนึ่ง
<span class="LineNumber"> 013 </span>ของข้าพเจ้ามีอยู่ ( ไม่อาจสอนท่านทั้งหลายต่อไปได้ ) เชิญท่าน

<span class='footnote'>เสนาสนะซ้ำกับอาวาส  น่าจะหมายเอาปัจจัย ๓ เท่านั้น</span>
<span class='footnote'>        ที่ท่านแก้ไว้นั้น  ดูจะจำกัดว่า  ลาภนั้นเป็นปลิโพธแก่ภิกษุผู้มีบุญเท่านั้น  หาเป็น</span>
<span class='footnote'>ปลิโพธแก่ภิกษุผู้มีบุญไม่   ซึ่งความจริงไม่น่าจะจำกัดเช่นนั้น  เพราะแม้ภิกษุผู้ไม่มีบุญ</span>
<span class='footnote'>ใครถวายอะไรนิดหน่อย แปลว่ามีลาภน้อย  ก็อาจติดใจข้องอยู่ในลาภนิด ๆ หน่อย ๆ</span>
<span class='footnote'>นั้น เกิดเป็นกังวลขึ้น จนไม่อาจตัดไปได้  เช่นนี้จะไม่จัดเป็นลาภปลิโพธหรือ แต่ก็อาจ</span>
<span class='footnote'>เป็นได้ว่าท่านมุ่งแสดงเป็นตัวอย่างว่า  ลาภเกิดเป็นปลิโพธได้อย่างไรเท่านั้น หาได้มุ่ง</span>
<span class='footnote'>จำกัดความเช่นว่าไม่</span>
<span class='footnote'>        ที่ว่า ถวายปัจจัยของบริวารมากนั้น มหาฎีกาให้ความหมายว่า เจตนามุ่งถวาย</span>
<span class='footnote'>ปัจจัยอย่างหนึ่ง  แต่ถวายของอื่น ๆ ให้เป็นบริวารปัจจัยนั้นไปด้วย  เช่นถวายบิณฑบาต</span>
<span class='footnote'>ก็ถวายสบงจีวรและของใช้อื่น ๆ เป็นบริวารบิณฑบาตไปด้วย นี้ก็เป็นอย่างที่เราทำอยู่</span>
<span class='footnote'>ทุกวันนี้  คำถวายทานทานต่าง ๆ ก็มีคำเช่นนี้แสดงไว้ด้วย เช่น อิมานิ มยํ ภนฺเต สปริวารานิ</span>
