<span class="LineNumber"> 001 </span>สวัสติกะสามเฉก<SUP>*</SUP> ( ที่หัวมัน ) แล้วสิ้นสงสัยความวางเฉยในการที่จะ
<span class="LineNumber"> 002 </span>เฟ้นว่างูไม่ใช่งูก็ย่อมมีฉันใด เมื่อเห็นลักษณะ ๓  ด้วยวิปัสสนา   
<span class="LineNumber"> 003 </span>ญาณแล้ว ความวางเฉยอันใดในอันที่จะเฟ้นความไม่เที่ยงแห่ง
<span class="LineNumber"> 004 </span>สังขารทั้งหลายเป็นต้นย่อมเกิดขึ้นแก่พระโยคาวจรผู้ทำวิปัสสนา ฉันนั้น
<span class="LineNumber"> 005 </span>เหมือนกัน ความวางเฉย ( ในการเฟ้นสังขาร ) นี้ ชื่อว่าวิปัสสนูเบกขา 
<span class="LineNumber"> 006 </span>ส่วนว่าเมื่อบุรุษนั้น ( กด )จับงูไว้มั่นด้วยไม้ง่ามแล้วคิดจะปล่อยมัน
<span class="LineNumber"> 007 </span>ไป ( โดย ) ไม่เบียดเบียนมันด้วย ไม่ให้มันกัดตัวด้วย ( ตรอง ) หา
<span class="LineNumber"> 008 </span>วิธีการจะปล่อยอยู่นั่นแล ความวางเฉยในอันจะจับเอา ( งู ) ไว้ย่อมมี
<span class="LineNumber"> 009 </span>ฉันใด เมื่อพระโยคาวจรเห็นภพ ๓ ดุจถูกไฟไหม้เพราะเห็นลักษณะ 
<span class="LineNumber"> 010 </span>๓ แล้ว ความวางเฉยอันใดในอันที่จะยึดถือเอาสังขารไว้ ย่อมมี
<span class="LineNumber"> 011 </span>ฉันนั้นแล ความวางเฉย ( ในการยึดสังขาร ) นี้ชื่อว่า สังขารุเบกขา 
<span class="LineNumber"> 012 </span>โดยประการดังนี้ เมื่อวิปัสสนูเบกขาสำเร็จแล้ว แม้สังขารุเบกขา
<span class="LineNumber"> 013 </span>ก็เป็นอันสำเร็จด้วยแท้ อันอุเบกขานี้แตกเป็น ๒ โดยกิจ กล่าวคือ 
<span class="LineNumber"> 014 </span>ความวางเฉยในความเฟ้น ( สังขาร ) และ ( วางเฉยใน ) 
<span class="LineNumber"> 015 </span>ความยึดถือ ( สังขาร ) ไว้ ( ดัง ) นี้แล 
<span class="LineNumber"> 016 </span>         ส่วนวิริยุเบกขาและเวทนูเบกขา ต่างกันและกันด้วย โดยความ
<span class="LineNumber"> 017 </span>ก็ต่างจากอุเบกขานอกนั้นด้วยแท้แล 
<span class="LineNumber"> 018 </span>         โดยนัยดังนี้ ในอุเบกขาเหล่านี้ ฌานุเบกขาท่านประสงค์เอา
<span class="LineNumber"> 019 </span>ในที่นี้ ฌานุเบกขานั้นมีความวางเฉยเป็นลักษณะ มีความไม่
