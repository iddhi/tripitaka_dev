<span class="LineNumber"> 001 </span>                   [อุปมาแห่งสัมมาวายามะ] 
<span class="LineNumber"> 002 </span>           ( ต่อไป ) นี้ เป็นคำแสดงความในสองคาถาข้างท้าย อุปมา 
<span class="LineNumber"> 003 </span>เหมือนผึ้ง ( ตัวหนึ่ง ) โง่ รู้ว่าดอกไม้ต้นโน้นบานแล้ว แล่นไปโดย
<span class="LineNumber"> 004 </span>ความเร็วจัด ( จนยั้งไม่อยู่ ) เลยล่วงดอกไม้นั้นไปเสีย ย้อนกลับมาถึง
<span class="LineNumber"> 005 </span>ต่อเมื่อเกสรมันสิ้น  ( รส ) เสียแล้ว อีกตัวหนึ่งก็โง่ แล่นไปโดยความ
<span class="LineNumber"> 006 </span>เร็วอ่อน ไปถึงเมื่อ ( เกสร ) มันสิ้น ( รส ) แล้วเหมือนกัน ส่วนผึ้งตัว
<span class="LineNumber"> 007 </span>ที่ฉลาดแล่นไปโดยความเร็วพอดี ถึงกลุ่มดอกไม้แล้ว เคล้าเอาเกสร
<span class="LineNumber"> 008 </span>เท่าที่ต้องการมาปรุงเป็นน้ำผึ้ง ก็ได้เสวยรสหวานสบายไป ฉันใด อนึ่ง
<span class="LineNumber"> 009 </span>อุปมาเหมือนเมื่อพวกลูกศิษย์ช่างสลักใช้ ( ปลาย ) มีดสลัก ( ลวดลาย ) 
<span class="LineNumber"> 010 </span>ลงที่ใบอุบล*ที่ลอยไว้ในอ่างน้ำกันอยู่ ศิษย์คนหนึ่งฉลาดเกินไปจด
<span class="LineNumber"> 011 </span>มีดลงเร็ว ( ทำให้ ) ใบอุบลฉีกบ้าง จมน้ำไปบ้างอีกคนหนึ่งไม่
<span class="LineNumber"> 012 </span>ฉลาดเลย ไม่กล้าแม้แต่จะเอามีดถู ( ใบอุบล ) เพราะกลัวมันฉีกและ
<span class="LineNumber"> 013 </span>จม ส่วนคนฉลาดแสดงรอยมีด ( เป็นลวดลาย ) ในใบอุบลนั้นด้วย
<span class="LineNumber"> 014 </span>ประโยค ( พยายาม ) ที่พอดี ก็เป็นผู้ได้ชื่อว่ามีศิลปะสำเร็จทำงานใน
<span class="LineNumber"> 015 </span>ตำแหน่งทั้งหลาย ( ที่ต้องใช้ศิลปะ ) เช่นนั้นย่อมได้ลาภ ฉันใด อนึ่ง 
<span class="LineNumber"> 016 </span>อุปมาเหมือนพระราชาทรงประกาศว่า" ผู้ใดนำใยแมลงมุมยาว
<span class="LineNumber"> 017 </span>ประมาณ ๔ วามา ( ถวาย ) ได้ ผู้นั้นจะได้ทรัพย์ ๔,๐๐๐  ( เหรียญ )

<span class='footnote'>*  ที่แปลทับศัพท์ว่า  ใบอุบล  ไม่แปลเป็นไทยว่าใบบัว  เพราะคำว่า  บัว  ในภาษาไทยเป็น</span>
<span class='footnote'>คำรวม  หมายถึงอุบลก็ได้  ปทุมก็ได้  แต่ในภาษาบาลี  อุบล  กับ  ปทุม   ไม่ใช่บัวพันธุ์</span>
<span class='footnote'>เดียวกัน  อุบลนั้นได้แก่บัวสาย  ปทุมได้แก่บัวก้าน ที่เราเรียกว่าบัวหลวง  ถ้าสีขาวเรียก</span>
<span class='footnote'>บุณฑริก  เพราะฉะนั้น  จึงแปลทับศัพท์ไว้เพื่อให้ทราบชนิดของมัน</span>
