<span class="LineNumber"> 001 </span>ดังกล่าวมานี้ ท่านผู้นี้ชื่อว่า ( กัมมฐานทายกะ ) ผู้ให้กรรมฐาน  
<span class="LineNumber"> 002 </span>พระโยคาวจร พึงเข้าไปหาท่านผู้ให้กรรมฐานนั้น 
<span class="LineNumber"> 003 </span>                                    [กัลยาณมิตร] 
<span class="LineNumber"> 004 </span>          คำว่า ผู้เป็นกัลยาณมิตร คือผู้เป็นมิตรที่ดี ผู้ตั้ง ( ตน ) อยู่ใน
<span class="LineNumber"> 005 </span>ฝ่ายเจริญ แสวงหาประโยชน์เกื้อกูล ( ผู้อื่น ) ไปฝ่ายเดียว กอบด้วยคุณ 
<span class="LineNumber"> 006 </span>มีคุณอย่างนี้ คือ 
<span class="LineNumber"> 007 </span>                     <B>น่ารัก ๑ น่าเคารพ ๑ น่ายกย่อง ๑ เป็น
<span class="LineNumber"> 008 </span>                     ผู้ว่ากล่าว ( คนอื่น ไม่เฉยเมย ) ๑ เป็นผู้ยอม
<span class="LineNumber"> 009 </span>                     ให้ ( คนอื่น ) ว่ากล่าว ( ไม่หัวดื้อ ) ๑ แต่ง 
<span class="LineNumber"> 010 </span>                     ( ธรรม ) ถกาอันลึกซึ้งได้ ๑ ไม่ชักชวน 
<span class="LineNumber"> 011 </span>                     ( ผู้อื่น ) ในที่อันไม่ควร ๑</B> 
<span class="LineNumber"> 012 </span>เป็นอาทิ ก็เพราะพระพุทธพจน์ว่า ดูกรอานนท์ จริงอยู่ สัตว์ทั้งหลาย 
<span class="LineNumber"> 013 </span>ผู้มีความเกิดเป็นธรรมดา อาศัยเราเป็นกัลยาณมิตรแล้ว ย่อม
<span class="LineNumber"> 014 </span>พ้นจากความเกิดได้ ดังนี้เป็นต้น พระสัมมาสัมพุทธเจ้าพระองค์เดียว 
<span class="LineNumber"> 015 </span>จึงเป็นกัลยาณมิตรที่ถึงพร้อมด้วยอาการทั้งปวง เพราะเหตุนั้น เมื่อ
<span class="LineNumber"> 016 </span>พระองค์ยังเสด็จอยู่ กรรมฐานที่ถือเอาในสำนักของพระผู้มีพระภาคเจ้า
<span class="LineNumber"> 017 </span>นั้นเท่านั้น จึงเป็นอันถือเอาอย่างดี แต่เมื่อพระองค์เสด็จปรินิพพานแล้ว 
<span class="LineNumber"> 018 </span>บรรดาพระมหาสาวก ๘๐ องค์ พระมหาสาวกองค์ใดยังทรงชีพอยู่ 
<span class="LineNumber"> 019 </span>ถือเอา ( กรรมฐาน ) ในสำนักพระมหาสาวกองค์นั้น จึงควรแม้นพระ
<span class="LineNumber"> 020 </span>มหาสาวกนั้นไม่มี ตนใคร่จะถือกรรมฐานบทใด ก็พึงถือเอาในสำนัก
<span class="LineNumber"> 021 </span>พระขีณาสพผู้ที่ทำจตุกฌานหรือปัญจกฌานให้เกิดด้วยอำนาจ
