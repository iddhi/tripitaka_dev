<span class="LineNumber"> 001 </span>ทีนี้นิมิตนั้นย่อมเป็น ( ปรากฏ ) แก่เธอ ดุจหนังโคตัวผู้ที่ถูกสับด้วยขอ 
<span class="LineNumber"> 002 </span>ตั้ง ๑๐๐ ( แผล ) ในเพราะที่สูงๆ ต่ำๆ เป็นแม่น้ำเป็นหล่มเลน เป็น 
<span class="LineNumber"> 003 </span>ภูเขาขรุขระแห่งแผ่นดินในที่ๆ ขยายไปๆ 
<span class="LineNumber"> 004 </span>           ก็แลพระอาทิกัมมิกะ ( โยคาวจร ) ผู้ได้ปฐมฌานในเพราะนิมิต
<span class="LineNumber"> 005 </span>นั้นแล้ว พึงเป็นผู้เข้า ( ฌาน ) ให้มาก ไม่พึงปัจจเวกขณ์มาก เพราะ
<span class="LineNumber"> 006 </span>เมื่อเธอปัจจเวกขณ์มากไป องค์ฌานทั้งหลายจะปรากฏเป็นหยาบ
<span class="LineNumber"> 007 </span>และมีกำลังเพลา<SUP>*</SUP> ทีนี้ เพราะปรากฏเสียอย่างนั้นแล้ว องค์ฌาน
<span class="LineNumber"> 008 </span>เหล่านั้นก็จะไม่สำเร็จเป็นปัจจัยแห่งความขวนขวายในเบื้องสูงของเธอ 
<span class="LineNumber"> 009 </span>( ต่อไป ) เธอขวนขวายไปในฌานอันไม่คล่องแคล่ว ( ในการเข้า ก็
<span class="LineNumber"> 010 </span>กลับ ) จะเสื่อมจากปฐมฌาน และไม่อาจบรรลุทุติยฌานได้เสียด้วย 
<span class="LineNumber"> 011 </span>เพราะเหตุนั้นพระผู้มีพระภาคเจ้าจึงตรัสไว้ว่า " ดูกรภิกษุทั้งหลาย
<span class="LineNumber"> 012 </span>เปรียบเหมือนแม่โคที่หากินบนภูเขา ( แต่มัน ) เป็นสัตว์โง่เซอะ ไม่รู้
<span class="LineNumber"> 013 </span>จักเขต ( ที่จะไปได้ไม่ได้ ) ไม่ฉลาดที่จะเที่ยวไปบนภูเขาอันขรุขระ 
<span class="LineNumber"> 014 </span>มันมีความรำพึงไปว่า ไฉนมันจะได้ไปทางทิศที่ไม่เคยไปบ้าง จะ
<span class="LineNumber"> 015 </span>ได้กินหญ้าที่ไม่เคยกินบ้าง จะได้ดื่มน้ำดื่มที่ไม่เคยดื่มบ้าง ดังนี้แล้ว
<span class="LineNumber"> 016 </span>มัน ( จะออกเดินไป ) ยังมิได้ตั้งเท้าหน้าให้มั่นยกเท้าหลังขึ้น มัน 
<span class="LineNumber"> 017 </span>( ก็จะพลาด ) ไม่ได้ไปทางทิศที่ไม่เคยไปด้วย ไม่ได้กินหญ้าที่ไม่เคย
<span class="LineNumber"> 018 </span>กินด้วย ไม่ได้ดื่มน้ำที่ไม่เคยดื่มด้วย ทั้งไม่ได้กลับไปสู่ที่ๆ มันยืน 
<span class="LineNumber"> 019 </span>รำพึงว่าไฉนมันจะได้ไปทางทิศที่ไม่เคยไป ฯลฯ  โดยสวัสดีด้วย นั่น
