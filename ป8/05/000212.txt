<span class="LineNumber"> 002 </span>ผู้มีชื่อเสียง )    อื่นแล้วจึงไป      ถามว่า      เพราะเหตุอะไร ?     ตอบว่า 
<span class="LineNumber"> 003 </span>เพราะหากว่าเมื่อเธอถูกอนิฏฐารมณ์ เช่นรูปร่างและเสียงแห่งสัตว์ร้าย 
<span class="LineNumber"> 004 </span>ทั้งหลายมีอมนุษย์ และสีห์เสือเป็นอาทิในป่าช้าครอบงำเอา ( จน ) 
<span class="LineNumber"> 005 </span>องคาพยพ ( หนาว )สั่นไปก็ดี<SUP>๑</SUP> ( ขวัญหนี ) ของที่บริโภคแล้วไม่อยู่ ( ในท้อง ) 
<span class="LineNumber"> 006 </span>ก็ดี<SUP>๒</SUP>  อาพาธเป็นอย่างอื่น ( อีก ) ก็ดี ( ไม่สามารถจะเดินกลับมาวิหาร
<span class="LineNumber"> 007 </span>ได้ ) ไซร้ เมื่อเป็นเช่นนั้น ท่านที่เธอบอกนั้นก็จัก ( ช่วย ) เก็บงำ
<span class="LineNumber"> 008 </span>บาตรจีวรของเธอไว้ให้อย่างดีในวิหารจักส่งพวกภิกษุหนุ่มบ้าง 
<span class="LineNumber"> 009 </span>สามเณรบ้าง ไปช่วยปรนนิบัติ ( พยาบาล ) ภิกษุนั้น อีกประการหนึ่ง 
<span class="LineNumber"> 010 </span>พวกโจรสำคัญว่า ป่าช้านั่นเป็นที่ไม่ต้องระแวง ( ว่าจะมีคนรู้เห็น ) 
<span class="LineNumber"> 011 </span>( มันไป ) ทำการมาแล้วก็ดี ยังมิได้ทำการก็ดี จึงมามั่วกันอยู่ ( ในนั้น )
<span class="LineNumber"> 012 </span> มัน ( ไปทำการมาแล้ว ) ถูกคนทั้งหลายตามมาทันเข้าก็จะทิ้งห่อของไว้
<span class="LineNumber"> 013 </span>ใกล้ๆ ภิกษุแล้วหนีไปเสียก็ได้คนทั้งหลาย ( เห็นของตกอยู่ใกล้ภิกษุ ) 
<span class="LineNumber"> 014 </span>ก็จะสำคัญว่า " พวกเราพบโจรพร้อมทั้งของกลาง " แล้วจับภิกษุไป
<span class="LineNumber"> 015 </span>ทรมาน ทีนั้นท่านที่เธอบอก ( ทราบเข้า ) ก็จัก ( ไปช่วย ) อ้อนวอน
<span class="LineNumber"> 016 </span>ว่า " ท่านทั้งหลายอย่าทรมานภิกษุนี้เลย เธอบอกข้าพเจ้าแล้วจึงไป
<span class="LineNumber"> 017 </span>ทำงานอันนี้ดอก " ( ชี้แจง ) ให้คนเหล่านั้นเข้าใจถูกแล้ว ทำความ
