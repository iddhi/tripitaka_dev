<span class="LineNumber"> 002 </span>พิถีพิถันใช้อะไรก็ได้ ก็พึงน้อมถวายของตามแต่จะได้เถิด จะว่ากันมาก 
<span class="LineNumber"> 003 </span>ไปไย สัมมาวัตร ( วัตรอันชอบ ) ที่พระผู้มีพระภาคเจ้าทรงบัญญัติ
<span class="LineNumber"> 004 </span>ไว้แล้วในมหาขันธกะว่า ดูกรภิกษุทั้งหลาย อันเตวาสิก พึงทำวัตร
<span class="LineNumber"> 005 </span>อันชอบในอาจารย์ นี้เป็นการทำวัตรชอบในข้อนั้น คือ อันเตวาสิก 
<span class="LineNumber"> 006 </span>พึงลุกขึ้นแต่เช้าตรู่ ถอดรองเท้า ทำอุตตราสงค์เฉวียงบ่าแล้ว ( เข้าไป ) 
<span class="LineNumber"> 007 </span>ถวายไม้ชำระฟัน ถวายน้ำล้างหน้า ปูอาสนะ ถ้ายาคูมีอยู่ พึงล้าง
<span class="LineNumber"> 008 </span>ภาชนะ น้อมยาคูเข้าไปถวาย ดังนี้เป็นต้นนั้นอันใด พึงทำสัมมาวัตร
<span class="LineNumber"> 009 </span>นั้นทุกข้อเถิด 
<span class="LineNumber"> 010 </span>          อันศิษย์ผู้ยังครูให้พอใจอยู่ด้วยวัตรสมบัติอย่างนี้ ในเวลาเย็น
<span class="LineNumber"> 011 </span>ท่านปล่อยให้ ( กลับ ) ไป จึงกราบ ( ลา )ไป เมื่อใดท่านถามว่า
<span class="LineNumber"> 012 </span>เธอมาเพื่ออะไร จึงเรียนเหตุแห่งการมาเมื่อนั้น ถ้าท่านไม่ถามเสียเลย 
<span class="LineNumber"> 013 </span>แต่ว่าท่านยินดี( การทำ ) วัตรไซร้ เมื่อล่วง ๑๐ วัน หรือปักข์หนึ่ง
<span class="LineNumber"> 014 </span>ไปแล้ว ( ได้โอกาส ) ณ วันหนึ่ง แม้ท่านปล่อย ( ในตอนเย็น )แล้ว
<span class="LineNumber"> 015 </span>ก็อย่าไป ให้ท่านทำโอกาสแล้ว เรียนบอกเหตุแห่งการมา หรือว่า
<span class="LineNumber"> 016 </span>เธอแสร้งไป ( หาท่าน ) ผิดเวลา ( เพื่อจะให้ ) ท่านถามว่า มาทำไม 
<span class="LineNumber"> 017 </span>จึงกราบเรียน ( เหตุแห่งการมาให้ท่านทราบ ) ถ้าท่านสั่งให้มาตอนเช้า
