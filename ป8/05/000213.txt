<span class="LineNumber"> 001 </span>                    [วิธีไปดูอสุภตามนัยอรรถกถา] 
<span class="LineNumber"> 002 </span>          เพราะเหตุนั้น พระโยคาวจรผู้ได้บอกแก่ภิกษุมีประการดังกล่าว
<span class="LineNumber"> 003 </span>แล้ว เกิดความจำนงยิ่งในอันจะดูอสุภนิมิตพึงยังปีติโสมนัสให้เกิดขึ้น<SUP>*</SUP>
<span class="LineNumber"> 004 </span>ไป ( ดู ) โดยวิธีที่ท่านกล่าวไว้ในอรรถกถาทั้งหลาย ( ให้ ) เหมือน
<span class="LineNumber"> 005 </span>กษัตริย์เกิดปีติโสมนัสเสด็จไปสู่สถานที่อภิเษก เหมือนผู้จะบูชายัญ 
<span class="LineNumber"> 006 </span>เกิดปีติโสมนัสไปสู่โรงบูชายัญ หรืออนึ่ง เหมือนคนไม่มีทรัพย์เกิดปีติ
<span class="LineNumber"> 007 </span>โสมนัสไปสู่ที่ตั้งขุมทรัพย์ฉะนั้นเถิด จริงอยู่ คำ ( ต่อไป ) นี้ ท่าน
<span class="LineNumber"> 008 </span>กล่าวไว้ ( ในอรรถกถา ) ว่า " ภิกษุเมื่อจะขึ้นเอาอสุภนิมิตชนิดอุทธุ-
<span class="LineNumber"> 009 </span>มาตกะ ย่อมไปผู้เดียว ไม่ ( ต้อง ) มีสหาย มีสติตั้งมั่นไม่หลงลืม มี
<span class="LineNumber"> 010 </span>อินทรีย์ทั้งหลายอยู่กับตัว ( คือสำรวม ) มีความคิดไม่ไปนอกตัว ( คือ
<span class="LineNumber"> 011 </span>ไม่คิดกลัวอะไรไปต่างๆ ) พิจารณาดูทางไปมาไว้ อสุภนิมิตชนิด
<span class="LineNumber"> 012 </span>อุทธุมาตกะมีทอดอยู่ที่แห่งใด เธอย่อมทำก้อนหินก็ดี จอมปลวกก็ดี 
<span class="LineNumber"> 013 </span>ไม้ต้นก็ดี ไม้กอก็ดี ไม้เถาก็ดี ( อันมีอยู่ )ในที่แห่งนั้น ให้เป็นนิมิต
<span class="LineNumber"> 014 </span>ร่วม ย่อมทำ ( วัตถุนั้นๆ ) ให้เป็นอารมณ์ร่วม ครั้นทำ ( วัตถุนั้นๆ ) 
<span class="LineNumber"> 015 </span>ให้เป็นนิมิตร่วมแล้ว ทำ ( วัตถุนั้นๆ ) ให้เป็นอารมณ์ร่วมแล้ว  จึง
<span class="LineNumber"> 016 </span>เข้าไปกำหนดอุทธุมาตกอสุภนิมิตตามความที่มันเป็นของมันเอง ( โดย
<span class="LineNumber"> 017 </span>อาการ ๑๑ คือ ) โดยสีกายบ้าง โดยเพศบ้างโดยสัณฐานบ้าง โดย
<span class="LineNumber"> 018 </span>ทิศบ้าง โดยโอกาสบ้าง โดยตัดตอนบ้าง ( และ ) โดยที่ต่อ โดยช่อง 
<span class="LineNumber"> 019 </span>โดยที่เว้า โดยที่นูน โดยรอบตัว เธอย่อมทำนิมิตนั้นให้เป็นสิ่งที่
<span class="LineNumber"> 020 </span>ถือเอาไว้อย่างดีแล้ว ย่อมจดจำเอานิมิตนั้นให้เป็นสิ่งที่จำได้อย่าง

<span class='footnote'>*  คือเกิดปีติโสมนัสว่า  จักพ้นจากชรามาณะด้วยปฏิปทานี้แน่</span>
