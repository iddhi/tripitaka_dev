<span class="LineNumber"> 002 </span>ก็เป็นอสุภฉันนั้นเหมือนกัน เป็นแต่ลักษณะอสุภในสรีระคนเป็นนั้น 
<span class="LineNumber"> 003 </span>ไม่ปรากฏ เพราะเครื่องแต่งกายอันเป็นอาคันตุกะปิดบังไว้ โดยปกติ
<span class="LineNumber"> 004 </span>แล้ว อันสรีระนี้ มีกระดูก  ๓๐๐ ท่อนกว่าๆ เป็นโครง ( กระดูกนั้น ) 
<span class="LineNumber"> 005 </span>ชนติดกันด้วยข้อต่อ ๑๘๐ ข้อ ยึดกันอยู่ด้วยเอ็น ๙๐๐ เส้น ฉาบด้วยเนื้อ
<span class="LineNumber"> 006 </span>๙๐๐ ชิ้น หุ้มด้วยหนังสดบางๆ ปิดด้วย ( หนัง ) ผิว ( คือหนังกำพร้า ) 
<span class="LineNumber"> 007 </span>มีช่องทะลุปรุไป มีมันซึมออกทั้งข้างบนและข้างล่าง ดุจโถมันข้น
<span class="LineNumber"> 008 </span>เป็นบ่อนของหมู่หนอน เป็นบ่อเกิดแห่งโรคทั้งหลาย เป็นแหล่งแห่ง
<span class="LineNumber"> 009 </span>ทุกขธรรม ( ธรรมฝ่ายทุกข์ ) ทั้งหลาย เป็นที่หลั่งออกเนืองๆ ( แห่ง
<span class="LineNumber"> 010 </span>ของไม่สะอาด ) ทางปากแผลทั้ง ๙ ดุจฝีเรื้อรังที่แตกแล้ว คือเป็นสรีระ
<span class="LineNumber"> 011 </span>ที่มีมูลตาออกทางตาทั้งสองข้าง มูลหูออกทางช่องหู ( ทั้งสองข้าง ) 
<span class="LineNumber"> 012 </span>น้ำมูกออกทางช่องจมูก ( ทั้งสองข้าง )อาหาร ( สำรอก ) น้ำดี เสมหะ
<span class="LineNumber"> 013 </span>และเลือดออกทางปาก อุจจาระปัสสาวะออกทางทวารเบื้องล่าง 
<span class="LineNumber"> 014 </span>( แถม ) หยาดเหงื่ออันไม่สะอาด ออกทางขุมขน ๙๙,๐๐๐ ขุม ตัวแมลง
<span class="LineNumber"> 015 </span>ทั้งหลายมีแมลงวันหัวเขียวเป็นต้นรุมตอมไป บุคคลไม่ปรนนิบัติสรีระ
<span class="LineNumber"> 016 </span>ไรเล่า ด้วยสรีรกิจมีสีฟันและบ้วนปาก สระหัว อาบน้ำ นุ่งผ้า ห่มผ้า 
<span class="LineNumber"> 017 </span>เป็นต้น เป็นคน ( ปล่อยกาย )ตามกำเนิด มีผมหยาบยุ่งเที่ยวไปตาม
<span class="LineNumber"> 018 </span>หมู่บ้าน เป็นราชาก็ดี เป็นคนชั้นเลวมีรับจ้างเทขยะและคนจัณฑาล
<span class="LineNumber"> 019 </span>เป็นต้นผู้ใดผู้หนึ่งก็ดี ก็หามีความแปลกกันไม่ เพราะความที่มีสรีระ
