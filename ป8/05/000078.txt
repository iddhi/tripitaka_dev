<span class="LineNumber"> 004 </span>วิหารที่เหมาะสม . . . . " ดังนี้ต่อไป 
<span class="LineNumber"> 005 </span>          ก่อนอื่น เมื่อพระโยคีผู้ใดอยู่ในวิหารเดียวกันกับพระอาจารย์
<span class="LineNumber"> 006 </span>มีความผาสุก พระโยคีนั้น เมื่อจะชำระกรรมฐานก็พึงอยู่ในวิหาร
<span class="LineNumber"> 007 </span>นั้นแหละ ถ้า ( เมื่ออยู่ ) ในวิหารนั้นไม่มีความผาสุกไซร้ วิหาร
<span class="LineNumber"> 008 </span>อื่นใดเป็นสัปปายะมีอยู่ในที่ ( ไกลแต่วิหารของอาจารย์ ) ประมาณ
<span class="LineNumber"> 009 </span>คาวุตหนึ่งหรือกึ่งโยชน์ หรือแม้ประมาณถึงโยชน์หนึ่งก็ดี ก็พึง ( ไป ) 
<span class="LineNumber"> 010 </span>อยู่ในวิหารนั้นเถิด ด้วยเมื่อได้อยู่อย่างนั้น ครั้นมีความสนเท่ห์หรือ
<span class="LineNumber"> 011 </span>เกิดความหลงลืมในข้อกรรมฐานที่ตรงไหนก็ดี  จะได้ ( ลุกขึ้น ) 
<span class="LineNumber"> 012 </span>ทำวัตรในวิหาร ( ที่ตนอยู่ ) แต่เช้าแล้ว ( ออกเดิน ) เที่ยวบิณฑบาต 
<span class="LineNumber"> 013 </span>( ฉัน ) ในระหว่างทาง ไปถึงที่อยู่ของพระอาจารย์ในเวลา ( ท่าน ) 
<span class="LineNumber"> 014 </span>เสร็จภัตกิจพอดี แล้ววันนั้นชำระกรรมฐานในสำนักของพระอาจารย์
<span class="LineNumber"> 015 </span>เสีย ( ให้เสร็จ ) วันรุ่งขึ้นก็กราบ ( ลา ) พระอาจารย์ ( เดินทางกลับ ) 
<span class="LineNumber"> 016 </span>ไป เที่ยวบิณฑบาต ( ฉัน ) ในระหว่างทาง ยังไม่ทันเหนื่อยก็อาจ
