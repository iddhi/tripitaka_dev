<span class="LineNumber"> 003 </span>ของ ๆ เขานั้นแหละให้ ( เขา ) ถ้า ( ของ ๆ เขา ) ไม่มี จึงให้ของ ๆ  ตน
<span class="LineNumber"> 004 </span>เป็นของยืม  เมื่อ ( หา ) ได้ภายหลัง จึงถือเอา ( คืน ) เมื่อ ( หา ) ไม่ได้
<span class="LineNumber"> 005 </span>ก็อย่าทวงเลย จะทำยา จะให้ยาแก่สามีของพี่น้องหญิงผู้มิใช้ญาติ
<span class="LineNumber"> 006 </span>ย่อมไม่ควร แต่พึงให้แก่พี่น้องหญิง  บอก ( เขา ) ให้ให้แก่สามีของเขา
<span class="LineNumber"> 007 </span>นัยแม้ในภริยาของพี่น้องชาย  ก็ดุจนัยนี้  ส่วนบุตรทั้งหลายของเขา
<span class="LineNumber"> 008 </span>เป็นญาติของเธอแท้  เพราะฉะนั้น  ทำ ( ยา ) ให้บุตรทั้งหลายของเขา

<span class="LineNumber"> 010 </span>                                       [อาพาธปลิโพธ]

<span class="LineNumber"> 012 </span>จึงชื่อว่า  ปลิโพธ  เพราะเหตุนั้น พึงตัดมันเสียด้วยการทำเภสัช     แต่ถ้า
<span class="LineNumber"> 013 </span>แม้เมื่อภิกษุทำเภสัช  ( รักษาอยู่ ) สองสามวัน  อาพาธไม่สงบไซร้  ก็พึง
<span class="LineNumber"> 014 </span>ติเตียนอัตภาพว่า   ข้าไม่ใช้บ่าว   ไม่ใช่ลูกจ้างของเจ้า    ก็ข้าเลี้ยงเจ้า
<span class="LineNumber"> 015 </span>นี้แหละ  จึงได้รับทุกข์ในสังสารวัฏอัน  ( ยาวนาน ) ไม่ปรากฏเงื่อนต้น
<span class="LineNumber"> 016 </span>เงื่อนปลาย  ( นี้ ) ดังนี้แล้ว  ทำสมณธรรมเถิด
