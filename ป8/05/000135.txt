<span class="LineNumber"> 001 </span>ปาฐะเพียงเท่านี้แล้ว บัดนี้จะทรงแสดงองค์ที่พึงประกอบเข้า ( แห่ง  
<span class="LineNumber"> 002 </span>ปฐมฌานนั้น ) จึงตรัสคำว่า <B>" สวิตกฺกํ สวิจารํ "</B> เป็นต้น 
<span class="LineNumber"> 003 </span>          ในคำเหล่านั้น ความตรึกชื่อว่า วิตก มีคำอธิบายไว้ว่า ความ
<span class="LineNumber"> 004 </span>จรด ( จิต ) ลงไป วิตกนี้นั้น มีอันปักจิตลงไปในอารมณ์เป็นลักษณะ 
<span class="LineNumber"> 005 </span>มีอันจรดไปทั่วๆ จรดไปรอบๆ เป็นกิจ จริงอย่างนั้น ด้วยองค์
<span class="LineNumber"> 006 </span>คือ วิตกนั้น พระโยคาวจรได้ชื่อว่าทำอารมณ์ให้เป็นสิ่งที่วิตกจรดไปทั่ว
<span class="LineNumber"> 007 </span>แล้ว ให้เป็นสิ่งที่วิตกจรดไปรอบแล้ว ดังนี้ วิตกนั้นมีการนำจิตเข้ามา
<span class="LineNumber"> 008 </span>ในอารมณ์ได้เป็นเครื่องปรากฏ 
<span class="LineNumber"> 009 </span>          ความตรองชื่อว่า วิจาร มีคำอธิบายไว้ว่า ความ ( พิจารณา ) 
<span class="LineNumber"> 010 </span>ตาม ( วิตก ) ไป วิจารนี้นั้นมีอันตามฟั้นอารมณ์เป็นลักษณะ มีอัน
<span class="LineNumber"> 011 </span>ประกอบสหชาตธรรมในอารมณ์นั้นไว้เป็นกิจ มีอันตามผูกพันจิต
<span class="LineNumber"> 012 </span>ไว้เป็นเครื่องปรากฏ 
<span class="LineNumber"> 013 </span>         เมื่อวิตกและวิจารนั้น แม้ว่าแยกกันไม่ได้ในที่ไหนๆ<SUP>*</SUP>ก็ดี 
<span class="LineNumber"> 014 </span>( แต่บัณฑิตพึงทราบว่า ) วิตกก็คือความจรดจิตลงไปทีแรก   ดุจเสียง
<span class="LineNumber"> 015 </span>เคาะระฆัง เพราะอรรถว่าเป็นสภาพยังหยาบ ( กว่าวิจาร ) และเพราะ
<span class="LineNumber"> 016 </span>อรรถว่าเป็นสภาพไปหน้า ( วิจาร ) วิจารก็คือความตามผูกพันไป
<span class="LineNumber"> 017 </span>ดุจเสียงครวญแห่งระฆัง เพราะอรรถว่าเป็นสภาพสุขุม ( กว่าวิตก ) 
<span class="LineNumber"> 018 </span>และเพราะเป็นสภาพตามฟั้น ( อารมณ์ ) อนึ่ง ในสองอย่างนั้น วิตก 
<span class="LineNumber"> 019 </span>ยังมีแกว่ง เป็นการไหวแห่งจิตในเวลาแรกเกิดขึ้น ดุจการกระพือปีก

<span class='footnote'>*  กตฺถจิ  ในที่ไหน ๆ  มหาฎีกาว่าหมายเฉพาะในปฐมฌาน  และในปริตตจิตตุบาน</span>
<span class='footnote'>ทั้งหลาย  ไม่ใช่ทั่วไป</span>
