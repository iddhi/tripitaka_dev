<span class="LineNumber"> 001 </span>ในความข้อนี้ ท่านกล่าว ( เป็นคาถา ) ไว้ว่า 
<span class="LineNumber"> 002 </span>              <B>พระผู้มีพระภาคเจ้าเป็น ( ภคฺคราโค ) ผู้ทรง 
<span class="LineNumber"> 003 </span>              หักราคะ ( ภคฺคโทโส ) ผู้ทรงหักโทสะ 
<span class="LineNumber"> 004 </span>              ( ภคฺคโมโห ) ผู้ทรงหักโมหะ เป็นผู้หาอาสวะ
<span class="LineNumber"> 005 </span>              มิได้ บาปธรรมทั้งหลายพระองค์ทรงหักเสีย
<span class="LineNumber"> 006 </span>              ( สิ้น ) แล้ว เพราะเหตุนั้น บัณฑิตจึงถวาย
<span class="LineNumber"> 007 </span>              พระนามว่า พระภควา</B> 
<span class="LineNumber"> 008 </span>          ก็แลความถึงพร้อมแห่งพระรูปกายของพระองค์ ผู้ทรงไว้ซึ่งพระ
<span class="LineNumber"> 009 </span>บุญลักษณะ ( ลักษณะอันเกิดเพราะบุญ ) นับด้วย ๑๐๐ เป็นอันท่าน
<span class="LineNumber"> 010 </span>แสดงแล้วด้วยความที่ทรงเป็นผู้มีพระภาคย์ ( คือบุญบารมี ) ความถึง
<span class="LineNumber"> 011 </span>พร้อมแห่งพระธรรม เป็นอันแสดงด้วยความที่ทรงเป็น ( ภคฺคโทสะ ) 
<span class="LineNumber"> 012 </span>ผู้หักโทสะ ความเป็นผู้ที่ชาวโลกและคนใกล้เคียงทั้งหลายรู้จักมากก็ดี 
<span class="LineNumber"> 013 </span>ความเป็นผู้ที่คฤหัสถ์และบรรพชิตทั้งหลาย พึงไปมาหาสู่ก็ดี ความ
<span class="LineNumber"> 014 </span>เป็นผู้สามารถในอันช่วยขจัดทุกข์กายทุกข์ใจให้แก่เขาทั้งหลายผู้ไปหาก็ดี 
<span class="LineNumber"> 015 </span>ความเป็นผู้ทำอุปการะเขาด้วยอามิสทานและธรรมทานก็ดีความเป็น
<span class="LineNumber"> 016 </span>ผู้สามารถในอันยังเขาให้ประกอบพร้อมไปด้วยโลกิยสุข และโลกุตตร-
<span class="LineNumber"> 017 </span>สุขก็ดี ก็เป็นอันท่านแสดงแล้วด้วยพระคุณสมบัติ ๒ อย่างนั้น<SUP>*</SUP>
