<span class="LineNumber"> 003 </span>                                 [กัมมปลิโพธ] 
<span class="LineNumber"> 004 </span>	 นวกรรม ( การก่อสร้าง ) ชื่อว่า กัมมะ ( การงาน ) อันภิกษุผู้
<span class="LineNumber"> 005 </span>ทำนวกรรมนั้น จำต้องรู้ทัพสัมภาระที่คนงานมีช่างไม้เป็นต้นได้ไว้แล้ว 
<span class="LineNumber"> 006 </span>และยัง ( หา ) ไม่ได้ ต้องขวนขวาย ( ดูแล ) ในงานที่เขาทำเสร็จแล้ว
<span class="LineNumber"> 007 </span>และยังไม่ได้ทำผู้ทำนวกรรมต้องมีปลิโพธไปเสียทุกอย่างดังนี้แหละ 
<span class="LineNumber"> 008 </span>แม้กัมมปลิโพธนั้น ก็พึงตัดเสียด้วยอุบายดังต่อไปนี้ ถ้างานเหลือน้อย 
<span class="LineNumber"> 009 </span>พึงทำเสียให้เสร็จ ถ้าเหลือมาก หากเป็นนวกรรมของสงฆ์ไซร้ พึง
<span class="LineNumber"> 010 </span>มอบหมายแก่สงฆ์ หรือภิกษุผู้นำภาระ ( ในสังฆกิจ ) ทั้งหลาย<SUP>๑</SUP> หาก
<span class="LineNumber"> 011 </span>เป็นนวกรรมส่วนตัว พึงมอบแก่ผู้นำภาระของตน ไม่มีภิกษุผู้นำภาระ
<span class="LineNumber"> 012 </span>เช่นนั้น ก็พึงสละ( งาน ) ให้แก่สงฆ์แล้วไปเถิด 
<span class="LineNumber"> 013 </span>                                [อัทธานปลิโพธ] 
<span class="LineNumber"> 014 </span>        การเดินทาง  ชื่อว่า อัทธานะ ก็บรรพชาเปกขะของภิกษุใดมีอยู่ 
<span class="LineNumber"> 015 </span>ที่ไหนก็ดี ปัจจัยชนิดไรๆ อันภิกษุใดจะพึงได้อยู่ที่ไหนก็ดี ถ้าภิกษุ
<span class="LineNumber"> 016 </span>นั้นยังไม่ได้ ( พบ ) บรรพชาเปกขะและปัจจัยนั้น เธอไม่อาจหยุดยั้ง
<span class="LineNumber"> 017 </span>อยู่ได้ อันคมิกจิต ( ความคิดจะไป ) แม้แห่งภิกษุผู้เข้าป่าทำสมณธรรม
<span class="LineNumber"> 018 </span>อยู่ ก็เป็นของยากที่จะบรรเทาได้<SUP>๒</SUP> เพราะฉะนั้น เธอพึงไปจัดทำกิจนั้น
