<span class="LineNumber"> 001 </span>                                  เสสกสิณนิเทศ
<span class="LineNumber"> 002 </span>                                       [อาโปกสิณ] 
<span class="LineNumber"> 003 </span>          ต่อไปนี้เป็นวิตถารกถาในอาโปกสิณ ในลำดับแห่งปฐวีกสิณ 
<span class="LineNumber"> 004 </span>ก็แลพระโยคาวจรผู้ใคร่จะเจริญปฐวีกสิณ เป็นผู้นั่งให้สบายฉันใดเล่า 
<span class="LineNumber"> 005 </span>แม้ผู้จะเจริญอาโปกสิณก็พึงเป็นผู้นั่งให้สบายฉันนั้น แล้วถือเอานิมิต
<span class="LineNumber"> 006 </span>ให้น้ำเถิด บททั้งปวงว่า   <B>กเต วา  อกเต  วา</B>   ( นิมิตที่แต่งขึ้นก็ดี   ที่
<span class="LineNumber"> 007 </span>มิได้แต่งขึ้นก็ดี ) เป็นต้น บัณฑิตพึง ( นำมากล่าว ) ให้พิสดาร 
<span class="LineNumber"> 008 </span>( ตามนัยที่กล่าวแล้วในปฐวีกสิณ ) อนึ่ง บททั้งปวงนั้นพึง ( นำมา
<span class="LineNumber"> 009 </span>กล่าว ) ให้พิสดารในอาโปกสิณนี้ฉันใด ในกสิณทั้งปวงก็พึงให้พิสดาร
<span class="LineNumber"> 010 </span>ฉันนั้นเถิด เพราะว่าต่อนี้ไป ข้าพเจ้าจักไม่กล่าวคำ <B>( กเต วา ฯ เป ฯ 
<span class="LineNumber"> 011 </span>วิตฺถาเรตพฺพํ )</B> เพียงเท่านี้อีก กล่าวแต่คำที่แปลกออกไป 
<span class="LineNumber"> 012 </span>          แม้ในอาโปกสิณนี้ นิมิตในน้ำที่มิได้แต่ง คือ ( น้ำ ) ในสระ 
<span class="LineNumber"> 013 </span>ในบึง หรือในทะเล<SUP>๑</SUP> ในสมุทรก็ตาม ย่อมเกิดขึ้นได้แก่พระโยคาวจร
<span class="LineNumber"> 014 </span>ผู้มีบุญ มีอธิการได้สร้างไว้ในปางก่อนแท้ ดุจพระจูฬสิวเถร ฉะนั้น 
<span class="LineNumber"> 015 </span>ได้ยินมาว่า เมื่อท่านผู้นั้นละลาภสักการะ ตั้งใจว่าจัก ( ไป ) อยู่อย่างสงัด
<span class="LineNumber"> 016 </span>ขึ้นเรือที่ท่ามหาติตถะไปชมพูทวีป มองดู ( น้ำใน ) มหาสมุทร
<span class="LineNumber"> 017 </span>ในระหว่าง  ( ทาง ) กสิณนิมิตอันมีส่วนเปรียบแห่ง ( น้ำใน ) มหา-
<span class="LineNumber"> 018 </span>สมุทร<SUP>๒</SUP>นั้น     ได้เกิดขึ้น ( ส่วน )พระโยคาวจรผู้มิได้สร้างอธิการไว้
