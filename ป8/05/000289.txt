<span class="LineNumber"> 001 </span>บรรลุได้ อนึ่ง ศาสนธรรมนั้น ( เมื่อ ) บุคคลฟังอยู่ ก็นำความงามมา  
<span class="LineNumber"> 002 </span>ให้ได้โดยแท้ แม้ด้วยการฟัง โดยข่มนิวรณ์ได้( ในขณะฟัง ) เพราะ
<span class="LineNumber"> 003 </span>เหตุนั้น จึงชื่อว่างามในเบื้องต้น ( เมื่อ ) บุคคลปฏิบัติอยู่ ก็นำความ
<span class="LineNumber"> 004 </span>งามมาให้ได้แม้ด้วยการปฏิบัติ โดยนำความสุขในสมถะและสุขใน
<span class="LineNumber"> 005 </span>วิปัสสนาให้ เพราะเหตุนั้น จึงชื่อว่างามในท่ามกลาง และ ( เมื่อ )
<span class="LineNumber"> 006 </span>บุคคลปฏิบัติแล้วอย่างนั้น ครั้นผลแห่งการปฏิบัติสำเร็จแล้ว ก็ย่อม
<span class="LineNumber"> 007 </span>นำความงามมาให้แม้ด้วยผลแห่งการปฏิบัติโดยนำความเป็นตาที
<span class="LineNumber"> 008 </span>บุคคล ( ผู้คงที่ไม่หวั่นไหว ) มาให้ เพราะเหตุนั้น จึงชื่อว่างามในเบื้อง
<span class="LineNumber"> 009 </span>ปลายแล ศาสนธรรมชื่อว่า <B>สฺวากฺขาตะ</B> เพราะเป็นธรรมงามในเบื้อง
<span class="LineNumber"> 010 </span>ต้นงามในท่ามกลางและงามในเบื้องปลาย โดยนัยดังกล่าวมาฉะนี้ 
<span class="LineNumber"> 011 </span> [เพราะประกาศพรหมจรรย์พร้อมทั้งอรรถ ... พยัญชนะ ...] 
<span class="LineNumber"> 012 </span>          อนึ่ง พระผู้มีพระภาคเจ้านั้น เมื่อทรงแสดงธรรม ย่อมทรง
<span class="LineNumber"> 013 </span>ประกาศคือทรงแสดงโดยนัยต่างๆ ซึ่งศาสนพรหมจรรย์ และมรรค-
<span class="LineNumber"> 014 </span>พรหมจรรย์อันใด ศาสนพรหมจรรย์และมรรคพรหมจรรย์อันนั้น ชื่อ
<span class="LineNumber"> 015 </span>ว่า <B>สาตฺถะ</B> เพราะความถึงพร้อมแห่งอรรถ ชื่อว่า <B>สพฺยญฺชนะ</B> 
<span class="LineNumber"> 016 </span>เพราะความถึงพร้อมแห่งพยัญชนะตามสมควร 
<span class="LineNumber"> 017 </span>         ชื่อว่า <B>สาตฺถะ</B> เพราะประกอบพร้อมด้วยอรรถบท ( บทแสดง
<span class="LineNumber"> 018 </span>อรรถ ๖ ) คือ สังกาสนะ ( แสดงความโดยสังเขป ) ปกาสนะ ( เริ่ม
<span class="LineNumber"> 019 </span>แสดงเป็นหัวข้อ ) วิวรณะ ( ขยายความ ) วิภชนะ ( จำแนกความ ) 
<span class="LineNumber"> 020 </span>อุตตานีกรณะ ( ทำความให้ตื้น ) ปัญญัตติ ( แต่งความให้เข้าใจ ) 
<span class="LineNumber"> 021 </span>ชื่อว่า สพฺยญฺชนะ เพราะความถึงพร้อมแห่ง ( พยัญชนบท ๖ คือ )
