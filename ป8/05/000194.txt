<span class="LineNumber"> 001 </span>                                        [เตโชกสิณ] 
<span class="LineNumber"> 002 </span>          ฝ่ายพระโยคาวจรผู้ใคร่จะเจริญเตโชกสิณ พึงถือเอานิมิตในไฟ 
<span class="LineNumber"> 003 </span>ในเตโชกสิณนั้น สำหรับพระโยคาวจรผู้มีบุญมีอธิการได้สร้างไว้ 
<span class="LineNumber"> 004 </span>เมื่อถือเอานิมิตในไฟที่มิได้แต่ง แลดูเปลวไฟที่ตะเกียง ที่เตา ที่ๆ  
<span class="LineNumber"> 005 </span>ระบมบาตรหรือที่ไฟไหม้ป่าที่ใดที่หนึ่งก็ตาม นิมิตก็เกิดขึ้นได้ ดังพระ
<span class="LineNumber"> 006 </span>จิตตคุตตเถร ฉะนั้น เล่ากันว่าเมื่อท่านผู้นั้นเข้าไปสู่โรงอุโบสถในวัน
<span class="LineNumber"> 007 </span>ธรรมสวนะ แลดูเปลวตะเกียงเท่านั้น นิมิตก็เกิดขึ้น ส่วนพระ
<span class="LineNumber"> 008 </span>โยคาวจรนอกนี้ ( คือผู้ไม่มีบุญญาธิการ ) ต้องแต่ง( กสิณนิมิต ) ขึ้น 
<span class="LineNumber"> 009 </span>          นี้เป็นวิธีทำ ในการแต่ง ( กสิณนิมิต ) นั้น พึงผ่าฟืนไม้แก่นที่ 
<span class="LineNumber"> 010 </span>( แห้ง ) สนิท* ผึ่ง ( แดด ) ไว้แล้ว ( ตัด ) ทำเป็นท่อนๆ ( หอบ ) ไปสู่
<span class="LineNumber"> 011 </span>โคนไม้หรือมณฑปที่เหมาะสม กอง ( ฟืน ) เข้าโดยอาการดุจระบมบาตร 
<span class="LineNumber"> 012 </span>ก่อไฟขึ้น แล้วทำให้เป็นช่อง ( กว้าง ) ประมาณคืบ ๔ นิ้ว ( เป็นวงกลม ) 
<span class="LineNumber"> 013 </span>เข้าที่เสื่อลำแพนหรือแผ่นหนังหรือผืนผ้าก็ได้ ตั้งเสื่อลำแพนหรือ
<span class="LineNumber"> 014 </span>แผ่นหนังหรือผืนผ้านั้นไว้ข้างหน้า นั่งโดยนัยที่กล่าวแล้วนั้นแล อย่า
<span class="LineNumber"> 015 </span>ใส่ใจถึงหญ้าและไม้ข้างใต้ ( ช่อง ) หรือควันและเปลวข้างบน ( ช่อง ) 
<span class="LineNumber"> 016 </span>ถือเอานิมิตที่เปลวทึบตรงกลาง ( ช่อง ) ไม่ต้องพิจารณาสี ( ของไฟ ) 
<span class="LineNumber"> 017 </span>โดยว่ามันเป็นสีขาว หรือว่าสีเหลืองเป็นต้น ไม่ต้องใส่ใจลักษณะ 
<span class="LineNumber"> 018 </span>( ของเตโชธาตุ ) โดยว่ามันร้อน พึงทำให้มันเป็นสิ่งเสมอกันกับที่อาศัย 
<span class="LineNumber"> 019 </span>( ของมัน     คือนึกเสียว่าสีนั้นมันเป็นอันเดียวกันกับไฟ    ซึ่งที่อาศัย
